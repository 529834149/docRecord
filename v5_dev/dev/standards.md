# 一系列（长度）规范
说明：1汉字=2字符

|字段    |描述|
|:------|:---------|
|用户名   |6-20位字符（字母、数字、英文中的连词号-）|
|姓名     |2-10位汉字|
|昵称     |2-20位字符|
|笔名     |2-20位字符|
|专栏名   |2-20位字符|
|头衔     |2-100位字符|
|简介     |2-500位字符|
|地址     |2-200位字符|
|代表作品  |2-400位字符|
|手机号    |11位数字|
|密码     |6-16位字符(字母、数字、符号)，区分大小写|
|email    |邮箱规范|
|博客地址    |url规范|
|标题 |2-50字符|
|专栏内容 |1-5w字符|
|栏目 |1-20字符|
